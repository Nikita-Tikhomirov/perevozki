# Генерация SEO-страниц по всем направлениям из Минска

## Цель

Создать публичные страницы для всех сочетаний 24 запросов из `СЕО для ГЕНЕРАЦИИ.xlsx` и полного каталога направлений из Минска, показанного в DOCX и уже существующего на `perewozki.by`. Блок городов является перелинковкой между реальными страницами, а не формой заявки.

## Объём

- Город отправления: Минск — это дополнительно подтверждено SEO-специалистом 17.07.2026.
- Обратные страницы `город → Минск` пока не создаются.
- 118 уникальных городов назначения в шести региональных группах живого блока сайта.
- 24 запроса из строк 2–25 Excel.
- Итого: 2 832 публичные страницы.
- Пять вариантов Title, Description и первого абзаца независимо распределяются по страницам. Выбор детерминирован для сочетания «запрос + Минск + город», а три поля на одной странице получают разные номера вариантов.

Лида, ошибочно продублированная на живом сайте в Витебской и Гродненской областях, генерируется один раз в корректной Гродненской группе. Опечатка `БудаМинск - Кошелево` исправляется на `Буда-Кошелёво`.

## URL и перелинковка

Публичный alias строится без тестового префикса:

`<slug-запроса>-minsk-<slug-города>`

На каждой странице блок `#cities` содержит 118 обычных ссылок на страницы того же запроса для остальных направлений. Ссылки не преобразуются в кнопки и не открывают форму. Три колонки и порядок областей повторяют дизайн ТЗ.

## Контент

Сопоставление столбцов Excel остаётся прежним: B–F Title, G–K Description, L–P первый абзац, Q цена, R частные лица, S бизнес, T/U FAQ, V контактный заголовок, W галерея. Плейсхолдеры `{Город1}` и `{Город2}` заменяются значениями маршрута. Поля B–P миксуются только в пределах своих пяти утверждённых вариантов; для столбцов Q–W, где вариант один, дополнительный текст не придумывается.

Маршрутные фразы базового HTML также заменяются, чтобы на страницах других городов не оставалось текста про Узду. Неудобные падежные конструкции переформулируются через нейтральное `по маршруту Минск – Город`.

## MODX и безопасность

- Отдельный публичный шаблон без `noindex`.
- Ресурсы: `published=1`, `hidemenu=1`, `searchable=1`, `cacheable=1`.
- Каждый ресурс содержит маркер `SEO-GENERATED:seo-2026-production`.
- Перед первой записью проверяются все alias; чужие совпадения останавливают публикацию.
- Манифест сохраняется каждые 10 страниц и поддерживает безопасное продолжение.
- Старые 24 тестовые страницы и тестовый шаблон остаются `noindex`, но получают правильную полную перелинковку.

## Проверка

- 2 832 уникальных alias и отсутствие незаменённых плейсхолдеров;
- 118 реальных ссылок в `#cities` на каждой странице;
- отсутствие остатков `Узда` вне каталога на страницах других городов;
- HTTP 200 для опубликованных URL и ссылок;
- desktop/mobile QA, FAQ, формы, отсутствие горизонтального переполнения;
- unit-тесты и глобальный smoke harness.
